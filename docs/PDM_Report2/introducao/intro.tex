Esta fase do trabalho prático permite consolidar as matérias leccionadas no decorrer do semestre, nomeadamente os componentes \textit{Android} do tipo \textit{started service} e \textit{bounded service}.

É também utilizada a \textit{Android Interface Definition Languaguage}, AIDL, para comunicação entre processos. 

A comunicação entre \textit{threads} é efectuada através de mecanismos de \textit{messaging} intrínsecos ao sistema operativo.
