\section{Actividade \textit{UserInfo}}

A versão anterior da actividade \texttt{UserInfoActivity} fazia uma utilização pouco optimizada dos recursos do sistema operativo, nomeadamente na execução de tarefas assíncronas. Foi alterado o mecanismo de obtenção de informação do utilizador para usar uma \texttt{AsyncTask}, garantindo que não existem problemas derivados ao uso manual de fios de execução.
