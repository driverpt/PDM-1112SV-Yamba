\section{\texttt{StatusPublishingService}}

Serviço que publica os novos \textit{tweets} inseridos pelo utilizador. Suporta modo \textit{offline}, guardando os \textit{tweets} na base de dados local caso não exista conectividade.Com recurso a \textit{BroadcastReceivers} no \textit{YambaPDMApplication} volta a tentar enviar os \texit{tweets} assim que houver conectividade.
